# Loan Repayment Prediction System

This project predicts whether a bank customer will repay a loan using Machine Learning.

## Features
- LightGBM-based classification model
- Flask REST API
- Single & batch (CSV) predictions
- Prediction logging
- Responsive frontend (HTML/CSS/JS)

## Tech Stack
- Python 3.10
- Scikit-learn
- LightGBM
- Flask
- HTML / CSS / JavaScript

How to Run
-----------------
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
